# 📌 Project Overview

Amdox AI-Powered Task Optimizer is a Data Science and Machine Learning based system designed to improve workplace productivity and employee well-being through real-time emotion analysis.

The system detects employee emotions using:
- Text Sentiment Analysis
- Facial Expression Recognition
- Speech Emotion Detection

Based on detected emotions and stress levels, the system:
- Recommends suitable tasks
- Detects stress and burnout
- Alerts HR managers
- Provides team mood analytics
- Tracks historical emotional trends

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/Amdox-AI-Task-Optimizer.git
cd Amdox-AI-Task-Optimizer
```

## 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Application

```bash

```
